Proposito del Sitio

El proposito del sitio es enfocarnos en el aprendizaje del tema Etiquetas semánticas y CSS Grid.

Etiquetas semánticas

Estos elementos, denominados generalmente elementos semánticos sirven para dividir un documento en partes lógicas, indicando el tipo de contenido que se incluye en cada parte, representan por tanto secciones lógicas o componentes de una aplicación web o un documento: header, nav, article, section, aside, footer [9, 12, 16].

Diez, T., Domínguez, M. J., Martínez, J. J., & Sáenz, J. (2012). Creación de páginas Web accesibles con HTML5. Consultado el, 26.

CSS Grid

Es un layout que facilita el proceso de colocar y distribuir a lo largo de una pagina. Presenta un sistema de rejilla bidimensional para usar en CSS que permitiran posicionar las principales areas de una pagina o pequeños detalles de la interfaz de usuario.

Murillo Ruiz, J. Rediseño de un blog Wordpress implementando CSS GRID y PWA.
